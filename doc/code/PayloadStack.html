    PayloadStack(buffer)

    Stacking payloads in a buffer. When you recieve a message through the network, it's added to the stack. As mesages can come in partial chunks, they need to be buffered until we have recieved the complete message. Messages can also be recieved in a batch of multiple messages, requiring an instance to segregate the message stack.
